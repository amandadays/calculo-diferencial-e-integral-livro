=== Limite de uma funcão

Antes de definir o limite apresntamos a noção intuitiva deste.

.{zwsp}
====
Sejam as funções latexmath:[$f$] e latexmath:[$g$] definidas pelas regras de correspondências:
[latexmath]
++++
\[
f(x)== x+1,\quad x\not=1\quad \mbox{ e} \quad 
g(x)=\lheft\{
\begin{array}{ccl} 
x^2+1 & &\mbox{ se } x\not=1;\\
3 & &\mbox{ se } x=1
\end{array}
\right.
\]
++++
Observamos que latexmath:[$f(2)$] não esta definida, enquanto que latexmath:[$g(1)=3$]. Porém, o comportamento de ambas funções em uma vizinhança de latexmath:[$1$], excluindo-se o ponto latexmath:[$1$], é exatamente o mesmo e pode ser descrito da seguinte forma. 

Para valores de latexmath:[$x$] próximos a latexmath:[$1$], com latexmath:[$x\not=1$], os valores de latexmath:[$f(x)$] e latexmath:[$g(x)$] se aproximam ao numero latexmath:[$\ell=2$]. No caso da função latexmath:[$f$], dizemos que latexmath:[$2$] é o limite de latexmath:[$f(x)$] quando latexmath:[$x$] tende (ou se aproxima) a latexmath:[$2$] e denota-se por latexmath:[$\lim\limits_{x\rightarrow 1}f(x)=2$]. De forma semelhante, para a função latexmath:[$g$] dizemos que latexmath:[$2$] é o limite de latexmath:[$g(x)$] quando latexmath:[$x$] tende a latexmath:[$1$] e se denota por  latexmath:[$\lim\limits_{x\rightarrow 1} g(x)=2$].

Notamos que o limite de  latexmath:[$f$] quando latexmath:[$x$] tende a latexmath:[$1$] não depende de latexmath:[$f(1)$] (em este caso não existe), porém depende dos valores que a função latexmath:[$f$] toma quando latexmath:[$x$] é próximo a latexmath:[$1$].
====

Definição::  Seja latexmath:[$f:\mathbb{R}\rightarrow \mathbb{R}$] uma função e latexmath:[$a$] um ponto que não necessariamente pertence ao latexmath:[${\rm Dom}(f)$], porém que toda vizinhança de latexmath:[$a$] contenha pontos de latexmath:[${\rm Dom}(f)$]. Diz-se que o numero latexmath:[$L$] é o *limite de latexmath:[$f$] no ponto latexmath:[$a$] se para cada latexmath:[$\epsilon>0$] é posivel encontrar um latexmath:[$\delta>0$], que depende de latexmath:[$a$] e latexmath:[$\epsilon$], tal que
[latexmath]
++++
\[
x\in {\rm Dom}(f), x\not=a\,\,\mbox{ e }\,\, 0<|x-a|<\delta\,\Rightarrow\, |f(x)-L|<\epsilon
\]
++++
e em este caso se denota:
[latexmath]
++++
\[
\lim\limits_{x\rightarrow a}f(x)=L
\]
++++
e se lhe *latexmath:[$L$] é o limite de latexmath:[$f(x)$] quando latexmath:[$x$] tende para latexmath:[$a$]*.

[NOTE]
Usando a notação de vizinhanças:
[latexmath]
++++
\[
\lim\limits_{x\rightarrow a}f(x)=L\,\,\Leftrightarrow\,\, \forall \epsilon>0,\,\exists \delta>0\,/\, \forall x\in {\rm Dom}(f)\cap B(a;\delta), x\not=a\,\Rightarrow\, f(x)\in B(L;\epsilon)
\]
++++

O conceito de limite implica na ideia de que latexmath:[$f(x)$] pode se fazer tão próximo a latexmath:[$L$] como queiramos, com tal que latexmath:[$x$] se escolha suficientemente próximo a latexmath:[$a$].

.{zwsp}
==== 
Se latexmath:[$\lim\limits_{x\rightarrow 2} 4x-5=3$]. Que tão próximo de latexmath:[$2$] deve estar latexmath:[$x$] para que latexmath:[$|f(x)-3|<0,001$]?

*Solução* Fazendo latexmath:[$\epsilon=0,001$], queremos que latexmath:[$|f(x)-3|<0,001$]. Para encontrar um latexmath:[$\delta$] adequado, notamos que latexmath:[$|f(x)-3|=|4x-5 -3|=4|x-2|<0,001$], de esta ultima desigualdade, obtemos latexmath:[$|x-2|<0,00025$], isto nos diz que se latexmath:[$x$] latexmath:[$2$] em menos de latexmath:[$0,00025$] então latexmath:[$f(x)$] dista de latexmath:[$3$] em menos de latexmath:[$0,001$].

[NOTE]
====
.. Para verificar o limite pela definição, inicialmente devemos decompor o termo latexmath:[$f(x)-L$] numa expressão onde apareça o termo latexmath:[$|x-a|$], isto é latexmath:[$|f(x)-L|=|x-a||g(x)|$],
.. o próximo passo é escolher um valor inicial latexmath:[$\delta=\delta_1
$] com o intuito de limitar a expressão latexmath:[$|g(x)|$], de forma que se latexmath:[$ 0<|x-a|<\delta_1$] implique que encontremos um numero latexmath:[$K>0$] tal que latexmath:[$|g(x)|<K$], 
.. logo
[latexmath]
++++
\[
\mbox{se }\,\,0<|x-a|<\delta_1\,\Rightarrow\, |f(x)-L|=|x-a||g(x)|<|x-a|K
\]
++++
assim, ao fazer latexmath:[$ |x-a|K<\epsilon$], obtemos latexmath:[$|x-a|<\frac{\epsilon}{K}=\delta_2$]. 

.. Logo o valor adequado para latexmath:[$\delta$] é latexmath:[$\delta=\min\lheft\{\delta_1,\frac{\epsilon}{K} \right\}$]. Portanto, se latexmath:[$ 0<|x-a|<\delta$] implica latexmath:[$|f(x)-L|=|x-a||g(x)|<\epsilon$], o que prova que latexmath:[$\lim\limits_{x\rightarrow a}f(x)=L$].

*Observação*
.. Algumas recomendações quando provemos limites pela definição são:
... Considerar um valor inicial de latexmath:[$\delta$], isto é latexmath:[$\delta=\delta_1$] tal que latexmath:[$0<|x-a|<\delta_1$]. Como exemplo pode ser considerado latexmath:[$\delta_1=1$] ou outro número menor.

... Ao escolher o valor inicial latexmath:[$\delta_1$], devemos ter cuidado de que não exista uma assintota vertical de latexmath:[$g(x)$] no intervalo latexmath:[$(a-\delta_1, a+\delta_1)$].

... Ao limitar latexmath:[$g(x)$] para o valor latexmath:[$\delta_1$] devemos lembrar alguma propriedades das desigualdades e do valor absoluto:
... Se latexmath:[$0<|x-a|<\delta$] então latexmath:[$a-\delta<x<a+\delta $];
... Se latexmath:[$a<y<b$] então latexmath:[$|y|<\max\{|a|,|b|\}$];
... Se latexmath:[$a<y<b$] então latexmath:[$y^2<k^2$] onde latexmath:[$ k=\max\{|a|,|b|\}$];
... Dados latexmath:[$ 0<\widetilde{\delta}<\delta$]. Se latexmath:[$\delta$] verifica a definição de limite, então latexmath:[$\widetilde{\delta}$] também verifica a definição de limite.
====

.{zwsp}
====
Se latexmath:[$f(x)=2x^2-5x+2$], provemos que latexmath:[$\lim\limits_{x\rightarrow 3}f(x)=5$].

Solução::
Dado latexmath:[$\epsilon>0$], devemos provar que se latexmath:[$0<|x-3|<\delta\Rightarrow |f(x)-5|<\epsilon$]. Pela primeira observação, temos que considerar latexmath:[$|f(x)-5| == |2x^2-5x+2-5|=|2x^2-5x-3|=|x-3||2x+1|$]. Escolhemos latexmath:[$\delta_1=1$] e buscamos o número latexmath:[$K>0$] tal que latexmath:[$0<|x-3|<1\Rightarrow |2x+1|<K$]. De fato:
[latexmath]
++++
\[
\mbox{se }\,\,|x-3|<1\Rightarrow 2<x<4 \Rightarrow 4<2x<8 \Rightarrow 5<2x+1<9 \Rightarrow |2x+1|<9
\]
++++
Multiplicamos a última desigualdade pela expressão latexmath:[$|x-3|$] e obtemos latexmath:[$|x-3||2x+1|<9|x-3|$] daqui deduzimos que latexmath:[$9|x-3|<\epsilon$] quando latexmath:[$|x-3|<\frac{\epsilon}{9}$]. Em resumo, dado latexmath:[$\epsilon>0$], latexmath:[$\exists \delta=\min\lheft\{1,\frac{\epsilon}{9} \right\}$] tal que 
[latexmath]
++++
\[
0<|x-3|<\delta \Rightarrow |f(x)-5| == |x-3||2x+1|<9|x-3|<\epsilon
\]
++++  
Portanto latexmath:[$\lim\limits_{x\rightarrow 3}f(x)=5$].
====

.{zwsp}
====
Se latexmath:[$f(x)=\frac{x+3}{x-3}$], provemos que latexmath:[$\lim\limits_{x\rightarrow 5}f(x)=4$].

Solução::
Dado latexmath:[$\epsilon>0$], devemos provar que se latexmath:[$0<|x-5|<\delta\Rightarrow |f(x)-4|<\epsilon$]. Pela primeira observação, temos que considerar latexmath:[$|f(x)-4| == |\frac{x+3}{x-3}-4|=|\frac{-3(x-5)}{x-3}|=\frac{3|x-5|}{|x-3|}$]. Escolhemos latexmath:[$\delta_1=1$] e buscamos o número latexmath:[$K>0$] tal que latexmath:[$0<|x-5|<1\Rightarrow \frac{1}{|x-3|}<K$]. De fato:
[latexmath]
++++
\[
\mbox{se }\,\,0<|x-5|<1\Rightarrow 4<x<6 \Rightarrow 1<x-3<3 \Rightarrow \frac{1}{3}<\frac{1}{x-3}<1 \Rightarrow \frac{1}{|x-3|}<1
\]
++++
Multiplicamos a última desigualdade pela expressão latexmath:[$3|x-5|$] e obtemos:
[latexmath]
++++
\[
\frac{3|x-5|}{|x-3|}<3|x-5|<\epsilon \Rightarrow |x-5|<\frac{\epsilon}{3}=\delta_2
\]
++++
Em resumo, dado latexmath:[$\epsilon>0$], latexmath:[$\exists \delta=\min\lheft\{\delta_1=1,\delta_2=\frac{\epsilon}{3} \right\}$] tal que 
[latexmath]
++++
\[
0<|x-5|<\delta \Rightarrow |f(x)-4| == \frac{3|x-5|}{|x-3|}<3|x-5|<3\frac{\epsilon}{3}=\epsilon
\]
++++  
Portanto latexmath:[$\lim\limits_{x\rightarrow 5}f(x)=4$].

=== Propriedade dos limites

Inicialmente, apresentamos uma propriedade útil dos números reais:

Propriedade:: Seja latexmath:[$x\in \mathbb{R}$]. Se latexmath:[$|x|< \epsilon$] para todo latexmath:[$\epsilon>0$], então latexmath:[$
x=0$].
Na sequencia apresentamos resultados importantes na teoria dos limites.

Teorema (Unicidade do limite):: Se existe o limite de uma função, então o limite é único, isto é: se latexmath:[$\lim\limits_{x\rightarrow a} f(x) =L_1$] e latexmath:[$\lim\limits_{x\rightarrow a} f(x) =L_2$] então latexmath:[$L_1=L_2$].

Teorema:: Se latexmath:[$f$] e latexmath:[$g$] são funções tais que:
.. latexmath:[$f(x)\lheq g(x)$], latexmath:[$\forall x\in B(a;r)$] com latexmath:[$x\not== a$];
.. latexmath:[$\lim\limits_{x\rightarrow a} f(x) =L$] e latexmath:[$\lim\limits_{x\rightarrow a} g(x) =M$]. 
Então, latexmath:[$ L\lheq M$], isto é latexmath:[$\lim\limits_{x\rightarrow a} f(x) \lheq \lim\limits_{x\rightarrow a} g(x)$]. 

Teorema (Sanduíche):: Se [$f$], latexmath:[$g$] e latexmath:[$h$] são funções tais que:

.. latexmath:[$f(x)\lheq g(x)\lheq h(x)$], latexmath:[$\forall x\in B(a;r)$] com latexmath:[$x\not== a$];
.. latexmath:[$\lim\limits_{x\rightarrow a} f(x) =\lim\limits_{x\rightarrow a} h(x) =L$]. 
Então, latexmath:[$\lim\limits_{x\rightarrow a} g(x) =L$]. 

Teorema:: Seja [$f$], latexmath:[$g$] e latexmath:[$h$] duas funções tais que:

.. latexmath:[$\lim\limits_{x\rightarrow a} f(x) =0$];
.. latexmath:Existe [$M>0$] tal que latexmath:[$|f(x)|<M$], latexmath:[$\forall x\in B(a;r)$] com latexmath:[$x\not== a$]; 
Então, latexmath:[$\lim\limits_{x\rightarrow a} f(x)g(x) =0$]. 

=== Propriedades Operacionais do limite

Teorema::
 Seja [$f$], latexmath:[$g$] dua funções tais que latexmath:[$\lim\limits_{x\rightarrow a} f(x) =L$] e latexmath:[$\lim\limits_{x\rightarrow a} g(x) =M$], então:

.. latexmath:[$\lim\limits_{x\rightarrow a} c =c$], onde latexmath:[$c$] é uma constante;
.. latexmath:[$\lim\limits_{x\rightarrow a}\lheft[cf(x)\right] =c\lheft[\lim\limits_{x\rightarrow a}f(x)\right]=cL$], onde latexmath:[$c$] é uma constante;
.. latexmath:[$\lim\limits_{x\rightarrow a} f(x) \pm g(x)=\lim\limits_{x\rightarrow a} f(x) \pm \lim\limits_{x\rightarrow a} g(x) =L\pm M$];
.. latexmath:[$\lim\limits_{x\rightarrow a} f(x) \cdot g(x)=\lim\limits_{x\rightarrow a} f(x) \cdot \lim\limits_{x\rightarrow a} g(x) =L\cdot M$];
.. Se latexmath:[$M\not=0$] então latexmath:[$\lim\limits_{x\rightarrow a} \dfrac{1}{g(x)} == \dfrac{1}{\lim\limits_{x\rightarrow a} g(x)}=\dfrac{1}{M}$]; 
.. Se latexmath:[$M\not=0$] então latexmath:[$\lim\limits_{x\rightarrow a} \dfrac{f(x)}{g(x)} == \dfrac{\lim\limits_{x\rightarrow a} f(x)}{\lim\limits_{x\rightarrow a} g(x)}=\dfrac{L}{M}$]. 

Como consequência do resultado anterior temos

Corolário:: Se latexmath:[$\lim\limits_{x\rightarrow a} f_i(x) =L_i$], para latexmath:[$i=1,2,\ldots,n$], então:

.. latexmath:[$\lim\limits_{x\rightarrow a} \lheft[f_1(x)+f_2(x)+\ldots+f_n(x) \right]=L_1+L_2+\ldots+ L_n$];
.. latexmath:[$\lim\limits_{x\rightarrow a} \lheft[f_1(x)\cdot f_2(x)\cdot\ldots\cdot f_n(x) \right]=L_1\cdot L_2\cdot\ldots \cdot L_n$].

Corolário:: Se latexmath:[$\lim\limits_{x\rightarrow a} f(x) =L$] e latexmath:[$n\in \mathbb{Z}$] então latexmath:[$\lim\limits_{x\rightarrow a} [f(x)]^n == \lheft[\lim\limits_{x\rightarrow a} f(x)\right]^n =L^n$]. (Quando latexmath:[$n\lheq 0$], o número latexmath:[$L$] deve ser diferente de zero).

Corolário:: Se latexmath:[$f(x)=a_0 x^n + a_1 x^{n-1}+\ldots+a_n$], onde latexmath:[$a_0,a_1,\ldots,a_n$] são constantes, então latexmath:[$\lim\limits_{x\rightarrow p}(a_0 x^n + a_1 x^{n-1}+\ldots+a_n) =a_0 p^n + a_1 p^{n-1}+\ldots+p_n =f(p)$].


Teorema:: Se latexmath:[$\lim\limits_{x\rightarrow a} f(x) =L$] e uma das condições seguintes é verificada:
.. latexmath:[$L\geq 0$] e latexmath:[$n$] é qualquer inteiro positivo; ou
.. latexmath:[$L< 0$] e latexmath:[$n$] é qualquer inteiro positivo impar;

então [$\lim\limits_{x\rightarrow a} \sqrt[n]{f(x)} == \sqrt[n]{\lim\limits_{x\rightarrow a} f(x)}=\sqrt[n]{L} $]. 
 
.{zwsp}
====
Calcule os seguintes limites:

.. latexmath:[$ \lim\limits_{x\rightarrow 2} \lheft( 5x^2-3x+4\right)== 5\lheft(\lim\limits_{x\rightarrow 2}x^2\right)-3\lheft(\lim\limits_{x\rightarrow 2} x\right)+\lheft(\lim\limits_{x\rightarrow 2} 4\right)=5(4)-3(2)+4=18$]

.. latexmath:[$ \lim\limits_{x\rightarrow 18} \sqrt[4]{x-2}=\sqrt[4]{\lim\limits_{x\rightarrow 18}(x-2)} == \sqrt[4]{\lim\limits_{x\rightarrow 18}x -  \lim\limits_{x\rightarrow 18}2 }== \lim\limits_{x\rightarrow 18}\sqrt[4]{18-2}=\sqrt[4]{16}=2$].

.. latexmath:[$\lim\limits_{x\rightarrow 2}\sqrt[3]{\dfrac{2x^5-4x-2}{x^3-6}}=\sqrt[3]{\lim\limits_{x\rightarrow 2} \dfrac{2x^5-4x-2}{x^3-6}} == \sqrt[3]{\dfrac{\lim\limits_{x\rightarrow 2}(2x^5-4x-2)}{\lim\limits_{x\rightarrow 2}(x^3-6)}} == \sqrt[3]{\dfrac{54}{2}}=\sqrt[3]{27}=3$].
====
[NOTE]
====
.. Se ao calcular um limite da forma  latexmath:[$\lim\limits_{x\rightarrow a}\frac{f(x)}{g(x)}$], temos que latexmath:[$\lim\limits_{x\rightarrow a} f(x)=0$] e latexmath:[$\lim\limits_{x\rightarrow a} g(x)=0$], isto é, não é possível aplicar a propriedade do Teorema ??, diz-se que o limite é *indeterminado* e é da forma latexmath:[$\frac{0}{0}$]. Em este caso, deve-se fatorar no numerador e no denominador o termo latexmath:[$(x-a)$], logo simplificar o fator comum para calcular o limite.

.. No geral, as formas indeterminadas são:
[latexmath]
++++
\[
\frac{0}{0},\,\, \frac{\infty}{\infty},\,\, \infty -\infty,\,\, 0\cdot\infty,\,\, 0^0,\,\,1^{\infty}\,\, \mbox{ e } \,\,\infty^0
\]
++++
====

Nos casos acima, devemos usar certos processos o artifícios que permitam eliminar a indeterminação. 

.{zwsp}
====
.. Calcular latexmath:[$ \lim\limits_{x\rightarrow -4} \dfrac{x^2-16}{3x+12}$]

Solução:: Ao avaliar obtemos que o limite tem uma uma indeterminação da forma latexmath:[$\frac{0}{0}$], em este caso fatoramos tanto o numerador como o denominador o termo latexmath:[$(x+4)$] e logo simplificamos, isto é devido a que no processo do limite temos latexmath:[$x\neq -4$]. De fato:

latexmath:[$ \lim\limits_{x\rightarrow -4} \dfrac{x^2-16}{3x+12} ==  \lim\limits_{x\rightarrow -4} \dfrac{\cancel{(x+4)}(x-4)}{3\cancel{(x+4)}} == \lim\limits_{x\rightarrow -4} \dfrac{x-4}{3}== \dfrac{1}{3}\lim\limits_{x\rightarrow -4} (x-4)=-\dfrac{8}{3}$]. 


.. Calcular latexmath:[$ \lim\limits_{x\rightarrow 0} \dfrac{\sqrt{x+3}-\sqrt{3}}{x}$]

Solução:: O limite tem uma uma indeterminação da forma latexmath:[$\frac{0}{0}$], em este caso racionalizamos e logo simplificamos, isto é: 

Se latexmath:[$\lim\limits_{x\rightarrow 0} \dfrac{\sqrt{x+3}-\sqrt{3}}{x} == \lim\limits_{x\rightarrow 0}\dfrac{(\sqrt{x+3}-\sqrt{3})(\sqrt{x+3}+\sqrt{3})}{x(\sqrt{x+3}+\sqrt{3})}=\lim\limits_{x\rightarrow 0} \dfrac{(x+3)-3}{x(\sqrt{x+3}+\sqrt{3})} == \lim\limits_{x\rightarrow 0} \dfrac{1}{\sqrt{x+3}+\sqrt{3}}== \dfrac{1}{\sqrt{3}+\sqrt{3})}=\dfrac{1}{2\sqrt{3}}$].


.. Calcular latexmath:[$ \lim\limits_{x\rightarrow 0} \dfrac{\sqrt{x+3}-\sqrt{3}}{x}$]

Solução:: O limite tem uma uma indeterminação da forma latexmath:[$\frac{0}{0}$], em este caso racionalizamos e logo simplificamos, isto é: 

Se latexmath:[$\lim\limits_{x\rightarrow 0} \dfrac{\sqrt{x+3}-\sqrt{3}}{x} == \lim\limits_{x\rightarrow 0}\dfrac{(\sqrt{x+3}-\sqrt{3})(\sqrt{x+3}+\sqrt{3})}{x(\sqrt{x+3}+\sqrt{3})}=\lim\limits_{x\rightarrow 0} \dfrac{(x+3)-3}{x(\sqrt{x+3}+\sqrt{3})} == \lim\limits_{x\rightarrow 0} \dfrac{1}{\sqrt{x+3}+\sqrt{3}}== \dfrac{1}{\sqrt{3}+\sqrt{3})}=\dfrac{1}{2\sqrt{3}}$].
====

[NOTE] 
Para racionalizar precisaremos lembrar que:
+
[latexmath]
++++
\[
\begin{array}{lcr}
(a^n-b^n)&=&(a-b)\underbrace{(a^{n-1}+a^{n-2}b+a^{n-3}b^2+\ldots+ab^{n-2}+b^{n-1})}_{\mbox{{\bf fator racionalizante}}}\\
(a^n+b^n)&=&(a+b)\overbrace{(a^{n-1}-a^{n-2}b+a^{n-3}b^2-\ldots-ab^{n-2}+b^{n-1})}^{}
\end{array}
\]
++++

.{zwsp}
====
.. Calcular latexmath:[$ \lim\limits_{x\rightarrow 4} \dfrac{3-\sqrt{5+x}}{1-\sqrt{5-x}}$]

Solução:: O limite tem uma uma indeterminação da forma latexmath:[$\frac{0}{0}$], em este caso fazemos uma dupla racionalização, assim: 
[latexmath]
++++
\[
\begin{array}{rcl}
\lim\limits_{x\rightarrow 4} \dfrac{3-\sqrt{5+x}}{1-\sqrt{5-x}}& == & \lim\limits_{x\rightarrow 4} \dfrac{(1+\sqrt{5-x})(3-\sqrt{5+x})(3+\sqrt{5+x}) }{(1+\sqrt{5-x})(1-\sqrt{5-x})(3+\sqrt{5+x})}\\
      & =& \lim\limits_{x\rightarrow 4}\dfrac{(4-x)(1+\sqrt{5-x})}{3+\sqrt{5+x}(x-4)}== \lim\limits_{x\rightarrow 4}\dfrac{1+\sqrt{5-x}}{3+\sqrt{5+x}} == -\dfrac{1+1}{3+3}=-\dfrac{2}{6}== -\dfrac{1}{3}
\end{array}
\]
++++

.. Calcular latexmath:[$ \lim\limits_{x\rightarrow 0} \dfrac{\sqrt{1+x^2}-\sqrt[4]{1+x^4}}{x^2}$]

Solução:: O limite tem uma uma indeterminação da forma latexmath:[$\frac{0}{0}$], em este caso fazemos uma racionalização, em este caso latexmath:[$a^4-b^4=(a-b)(b^3+a^2b+ab^2+b^3)$], onde latexmath:[$a=\sqrt[4]{(1+x^2)^2}=\sqrt{1+x^2}$] e latexmath:[$b=\sqrt[4]{1+x^4}$], obtemos 
[latexmath]
++++
\[
\begin{array}{rcl}
\lim\limits_{x\rightarrow 0} \dfrac{\sqrt{1+x^2}-\sqrt[4]{1+x^4}}{x^2} &=& \lim\limits_{x\rightarrow 0} \dfrac{\sqrt[4]{(1+x^2)^2}-\sqrt[4]{1+x^4}}{x^2}\\
&=& \lim\limits_{x\rightarrow 0} \frac{\lheft(\sqrt[4]{(1+x^2)^2}-\sqrt[4]{1+x^4}\right)\lheft[\sqrt{(1+x^2)^3} +(1+x^2)\sqrt[4]{1+x^4} + \sqrt{1+x^2}\sqrt{1+x^4} + \sqrt[4]{(1+x^4)^3}\right]}{x^2\lheft[\sqrt{(1+x^2)^3} +(1+x^2)\sqrt[4]{1+x^4} + \sqrt{1+x^2}\sqrt{1+x^4} + \sqrt[4]{(1+x^4)^3}\right]}\\
&=& \lim\limits_{x\rightarrow 0} \frac{(1+x^2)^2 - (1+x^4)}{x^2\lheft[\sqrt{(1+x^2)^3} +(1+x^2)\sqrt[4]{1+x^4} + \sqrt{1+x^2}\sqrt{1+x^4} + \sqrt[4]{(1+x^4)^3}\right]}\\
&=& \lim\limits_{x\rightarrow 0} \frac{2x^2}{x^2\lheft[\sqrt{(1+x^2)^3} +(1+x^2)\sqrt[4]{1+x^4} + \sqrt{1+x^2}\sqrt{1+x^4} + \sqrt[4]{(1+x^4)^3}\right]}\\
&=&\frac{2}{1+1+1+1}=\frac{1}{2}
\end{array}
\]
++++



.. Calcular latexmath:[$ \lim\limits_{x\rightarrow 64} \dfrac{\sqrt{x}-8}{\sqrt[3]{x}-4}$]

Solução::  O limite tem uma uma indeterminação da forma latexmath:[$\frac{0}{0}$], em este caso poderíamos fazer uma dupla racionalização, porem seria muito operativo, por outro lado, as quantidades sub-radicais são as mesmas e tem diversas raies assim optamos fazer uma mudança de variável com o intuito de simplificar a expressão:

Se escolhe uma variável que seja igual à candidate sub-radical e o expoente de esta variável é o minimo comum múltiplo dos índices dos radicais. Em nosso caso:

Seja latexmath:[$y^6=x$] onde latexmath:[$ {\rm m.c.m}(2,3)=6$], desde que latexmath:[$y^6=x$] temos latexmath:[$\sqrt{x}=y^3$] e latexmath:[$\sqrt[3]{x}=y^2$]. Para latexmath:[$x=64$], latexmath:[$y^6=64 \Rightarrow y=2$]. Substituindo:
[latexmath]
++++
\[
\lim\limits_{x\rightarrow 64} \dfrac{\sqrt{x}-8}{\sqrt[3]{x}-4}== \lim\limits_{y\rightarrow 2} \dfrac{y^3-8}{y^2-4}=\lim\limits_{x\rightarrow 2} \dfrac{(y-2)(y^2+2y+4)}{(y-2)(y+2)}== \lim\limits_{x\rightarrow 2} \dfrac{y^2+2y+4}{y+2}== \dfrac{4+4+4}{2+2}=\dfrac{12}{4}=3.
\]
++++


.. Calcular latexmath:[$ \lim\limits_{x\rightarrow 0} \dfrac{x\sqrt[3]{x+1}+\sqrt[4]{x+1}-1}{x^2\sqrt[3]{x+1}+\sqrt[4]{x+1}-1}$]

Solução::  Mudando a variável latexmath:[$x+1=y^{{\rm m.c.m}(3,4)}=y^{12}$], desde que latexmath:[$x+1\geq 0$], se latexmath:[$
x\rightarrow 0$] então latexmath:[$y\rightarrow 1$]:
++++
\[
\begin{array}{rcl}
\lim\limits_{x\rightarrow 0} \frac{x\sqrt[3]{x+1}+\sqrt[4]{x+1}-1}{x^2\sqrt[3]{x+1}+\sqrt[4]{x+1}-1} & == & \lim\limits_{x\rightarrow 0} \frac{(y^{12}-1)y^4+y^3-1}{(y^{12}-1)y^4+y^3-1}\\
&=& \lim\limits_{x\rightarrow 0} \frac{(y-1)(y^{11}+y^{10}+\ldots +1)y^4+(y-1)(y^2+y+1)}{(y-1)^2(y^{11}+y^{10}+\ldots +1)^2 y^4 +(y-1)(y^2+y+1)}\\
&=& \lim\limits_{x\rightarrow 0} \frac{(y^{11}+y^{10}+\ldots +1)y^4+y^2+y+1}{(y-1)(y^{11}+y^{10}+\ldots +1)^2 y^4+ y^2+y+1}\\
&=&\frac{11+3}{3} =\frac{14}{3}.
\end{array}
\]
====

=== Limites laterais

Quando se calcula latexmath:[$\lim\limits_{x\rightarrow a} f(x)=L$], reduzimos o problema a encontrar o número latexmath:[$L$] para o qual se aproximam os valores de latexmath:[$f(x)$] quando latexmath:[$x$] tende para latexmath:[$a$], isto é aproximando-se por valores menores que latexmath:[$a$] (pela esquerda) ou por valores maiores que latexmath:[$a$] (pela direita).

%%%%%%%%colocar o gráfico do espinoza mudar x0=a, l maiusculas

Para o caso dos limites laterais é mais simplhe, devido a que depende do comportamento da função latexmath:[$f(x)$] quando latexmath:[$x$] se aproxima para latexmath:[$a$] somente pela esquerda ou somente pela direita do latexmath:[$x$].

Definição::   
Seja latexmath:[$f$] uma função definida no intervalo latexmath:[$(c,a)$],latexmath:[$c<a$]. Diz-se que o número latexmath:[$L_1$] é o *limite lateral de latexmath:[$f(x)$] quando latexmath:[$x$] tende para latexmath:[$a$] pela esquerda*, o qual é denotado por latexmath:[$\lim\limits_{x\rightarrow a^{-}} f(x)=L_1$], se
+
[latexmath]
++++
\[
\forall \epsilon>0,\, \exists \delta >0\,:\, 0<a-x<\delta\,\Rightarrow\,|f(x)-L_1|<\epsilon
\]
++++ 

Definição::   
Seja latexmath:[$f$] uma função definida no intervalo latexmath:[$(a,d)$],latexmath:[$a<d$]. Diz-se que o número latexmath:[$L_2$] é o *limite lateral de latexmath:[$f(x)$] quando latexmath:[$x$] tende para latexmath:[$a$] pela direita*, o qual é denotado por: latexmath:[$\lim\limits_{x\rightarrow a^{+}} f(x)=L_2$], se
+
[latexmath]
++++
\[
\forall \epsilon>0,\, \exists \delta >0\,:\, 0<x-a<\delta\,\Rightarrow\,|f(x)-L_2|<\epsilon
\]
++++ 

Teorema:: Se latexmath:[$f$] é uma função definida numa vizinhança do ponto latexmath:[$a$], e latexmath:[$L\in \mathbb{R}$], então se verifica
[latexmath]
++++
\[
\lim\limits_{x\rightarrow a} f(x)=L\,\Leftrightarrow\, \lim\limits_{x\rightarrow a^{-}} f(x)=L\,\,\mbox{ e }\,\,\lim\limits_{x\rightarrow a^{+}} f(x)=L
\]
++++
Em outras palavras, existe o limite de uma função se, e somente se, existem os limites laterais e são iguais.


[NOTE]
====
.. Não existe o latexmath:[$\lim\limits_{x\rightarrow a} f(x)$]  em os seguintes casos:
... Quando não exista algum dos limites laterais.
... Quando os limites laterais existem e são diferentes.

.. Quando a função latexmath:[$f$] é definida por partes para latexmath:[$ x<a$] e para latexmath:[$x>a$], para encontrar o latexmath:[$\lim\limits_{x\rightarrow a} f(x)$] é necessário calcular os seus respectivos limites laterais.
====

.{zwsp}
==== 
Calcular se existe latexmath:[$\lim\limits_{x\rightarrow 2} f(x)$], onde latexmath:[$f(x)=\lheft\{\begin{array}{ccl}
x^2 & &\mbox{se } x<2; \\
4 & &\mbox{se } x=2; \\
8-2x &&\mbox{se } x> 2. 
\end{array}\right.
$]
Solução:: A função latexmath:[$f$] tem diferentes regras de correspondência para latexmath:[$ x<2$] e latexmath:[$ x>2$]. Precisamos calcular os limites laterais:
[latexmath]
++++
\[
\lim\limits_{x\rightarrow 2^{-}} f(x)== \lim\limits_{x\rightarrow 2^{-}} x^2 =2^2=4\quad\mbox{ e }\quad\lim\limits_{x\rightarrow 2^{+}} f(x)==  \lim\limits_{x\rightarrow 2^{+}} 8-2x =8-4=4.
\]
++++
Ao comparar os dois limites laterais ambos existem e são iguais, então existe latexmath:[$\lim\limits_{x\rightarrow 2} f(x)=4$]
====


.{zwsp}
==== 
Calcular se existe latexmath:[$\lim\limits_{x\rightarrow 0} x\sqrt{\dfrac{1}{4x^2}-16}$].

Solução:: latexmath:[$\lim\limits_{x\rightarrow 0} x\sqrt{\dfrac{1}{4x^2}-16} == \lim\limits_{x\rightarrow 0} x\sqrt{\dfrac{1-64x^2}{4x^2}} == \lim\limits_{x\rightarrow 0} \dfrac{x\sqrt{1-64x^2}}{2|x|}$], assim temos que a função latexmath:[$f$] tem diferentes regras de correspondência para latexmath:[$ x<0$] e latexmath:[$ x>0$]. Precisamos calcular os limites laterais:
[latexmath]
++++
\[
\begin{array}{ccl}
\lim\limits_{x\rightarrow 0^{-}} \dfrac{x\sqrt{1-64x^2}}{2|x|} &=& \lim\limits_{x\rightarrow 0^{-}} \dfrac{x\sqrt{1-64x^2}}{-2x}== \lim\limits_{x\rightarrow 0^{-}} -\dfrac{\sqrt{1-64x^2}}{2}=-\dfrac{1}{2}\\
\lim\limits_{x\rightarrow 0^{+}} \dfrac{x\sqrt{1-64x^2}}{2|x|} &=& \lim\limits_{x\rightarrow 0^{+}} \dfrac{x\sqrt{1-64x^2}}{2x}== \lim\limits_{x\rightarrow 0^{+}} \dfrac{\sqrt{1-64x^2}}{2}=\dfrac{1}{2}
\end{array}
\]
++++
Ao comparar os dois limites laterais ambos existem porém não são iguais, portanto não existe latexmath:[$\lim\limits_{x\rightarrow 0} f(x)$].


.{zwsp}==== Calcular se existe latexmath:[$\lim\limits_{x\rightarrow \frac{7}{3}}\sqrt{|x|+\lheft\lfloor 3x \right\rfloor} $].

*Solução* Observamos que a função tem uma expressão com o máximo inteiro, precisamos analisar os limites laterais numa vizinhança do latexmath:[$\frac{7}{3}$]:

.. Se latexmath:[$2\lheq x< \frac{7}{3} \Rightarrow 6\lheq 3x < 7 \Rightarrow \lheft\lfloor 3x \right\rfloor=6$], então latexmath:[$\lim\limits_{x\rightarrow {\frac{7}{3}}^{-}}\sqrt{|x|+\lheft\lfloor 3x \right\rfloor} =\lim\limits_{x\rightarrow {\frac{7}{3}}^{-}}\sqrt{x+6}=\dfrac{5\sqrt{3}}{3}$].

.. Se latexmath:[$\frac{7}{3}\lheq x< \frac{8}{3} \Rightarrow 7\lheq 3x < 8 \Rightarrow \lheft\lfloor 3x \right\rfloor=7$], então latexmath:[$\lim\limits_{x\rightarrow {\frac{7}{3}}^{+}}\sqrt{|x|+\lheft\lfloor 3x \right\rfloor} =\lim\limits_{x\rightarrow {\frac{7}{3}}^{+}}\sqrt{x+7}=\sqrt{\dfrac{28}{3}}$].

Ao comparar os dois limites laterais ambos existem porém não são iguais, portanto não existe latexmath:[$\lim\limits_{x\rightarrow \frac{7}{3}}\sqrt{|x|+\lheft\lfloor 3x \right\rfloor}$].

=== limites ao infinito

Consideremos a função latexmath:[$ f(x)=3+\dfrac{1}{x-3}$] com gráfico:
%%%%inserir o gráfico do espinoza pag 414 mudando 2 para 3

Analisando o gráfico para valores de latexmath:[$x$] cada vez mais grandes, o valor da função latexmath:[$f$] se aproxima a latexmath:[$3$], assim podemos afirmar que latexmath:[$\lim\limits_{x\rightarrow +\infty}f(x)=3$] quando latexmath:[$x$] decresce sem limite, o valor se aproxima para latexmath:[$3$]. Logo afirmamos que latexmath:[$\lim\limits_{x\rightarrow -\infty}f(x)=3$]. A estos tipos de limites são chamados limites ao infinito.

A seguir daremos as definições correspondentes:

Definição::  Seja latexmath:[$f:(a,+\infty)\rightarrow \mathbb{R}$] e latexmath:[$ L\in\mathbb{R}$]. Diz-se que latexmath:[$L$] é o limite de latexmath:[$f(x)$] quando latexmath:[$x$] tende para latexmath:[$+\infty$], e o denotamos por latexmath:[$\lim\limits_{x\rightarrow +\infty}f(x)=L$], se
[latexmath]
++++
\[
\forall \epsilon>0,\, \exists N>0\,:\, x>N\,\Rightarrow\,|f(x)-L|<\epsilon.
\]
++++


Definição::  Seja latexmath:[$f:(-\infty,a)\rightarrow \mathbb{R}$] e latexmath:[$ L\in\mathbb{R}$]. Diz-se que latexmath:[$L$] é o limite de latexmath:[$f(x)$] quando latexmath:[$x$] tende para latexmath:[$-\infty$], e o denotamos por latexmath:[$\lim\limits_{x\rightarrow -\infty}f(x)=L$], se
[latexmath]
++++
\[
\forall \epsilon>0,\, \exists M>0\,:\, x<-M\,\Rightarrow\,|f(x)-L|<\epsilon.
\]
++++

*Teorema* Seja latexmath:[$n$] um inteiro positivo quaisquer, então
.. latexmath:[$\lim\limits_{x\rightarrow +\infty}\dfrac{1}{x^n}=0$]
.. latexmath:[$\lim\limits_{x\rightarrow -\infty}\dfrac{1}{x^n}=0$].

*Teorema* Seja [$f$], latexmath:[$g$] dua funções definidas nos intervalos latexmath:[$(a,+\infty)$] e latexmath:[$(b,+\infty)$] respectivamente. Se latexmath:[$\lim\limits_{x\rightarrow +\infty} f(x) =L$] e latexmath:[$\lim\limits_{x\rightarrow +\infty} g(x) =M$], então:
.. latexmath:[$\lim\limits_{x\rightarrow +\infty} c =c$], onde latexmath:[$c$] é uma constante;
.. latexmath:[$\lim\limits_{x\rightarrow +\infty}\lheft[cf(x)\right] =c\lheft[\lim\limits_{x\rightarrow +\infty}f(x)\right]=cL$], onde latexmath:[$c$] é uma constante;
.. latexmath:[$\lim\limits_{x\rightarrow +\infty} f(x) \pm g(x)=\lim\limits_{x\rightarrow +\infty} f(x) \pm \lim\limits_{x\rightarrow +\infty} g(x) =L\pm M$];
.. latexmath:[$\lim\limits_{x\rightarrow +\infty} f(x) \cdot g(x)=\lim\limits_{x\rightarrow +\infty} f(x) \cdot \lim\limits_{x\rightarrow +\infty} g(x) =L\cdot M$];
.. Se latexmath:[$M\not=0$] então latexmath:[$\lim\limits_{x\rightarrow +\infty} \dfrac{1}{g(x)} == \dfrac{1}{\lim\limits_{x\rightarrow +\infty} g(x)}=\dfrac{1}{M}$]; 
.. Se latexmath:[$M\not=0$] então latexmath:[$\lim\limits_{x\rightarrow +\infty} \dfrac{f(x)}{g(x)} == \dfrac{\lim\limits_{x\rightarrow +\infty} f(x)}{\lim\limits_{x\rightarrow +\infty} g(x)}=\dfrac{L}{M}$]. 

*Observação*
.. Quando latexmath:[$x\rightarrow-\infty$], obtemos propriedades semelhantes às apresentadas no resultado anterior.
.. A forma mais pratica de calcular os limites quando latexmath:[$ x\rightarrow +\infty$] ou latexmath:[$ x\rightarrow -\infty$] de uma função racional, é dividendo tanto o numerador como o denominador, entre a maior potencia de latexmath:[$x$] do divisor que aparece na expresso dada, logo se aplica o critério do primeiro teorema.

.{zwsp}==== Calcule os seguintes limites:
.. latexmath:[$\lim\limits_{x\rightarrow +\infty}\dfrac{7x^2-8x+2}{5x^2+3x-3}$]
*Solução* Pela observação anterior, dividimos o numerador e o denominador por latexmath:[$x^2$] (maior potencia do denominador) e obtemos:
[latexmath]
++++
\[
\lim\limits_{x\rightarrow +\infty}\dfrac{7x^2-8x+2}{5x^2+3x-3}== \lim\limits_{x\rightarrow +\infty}\dfrac{7-\dfrac{8}{x}+\dfrac{2}{x^2}}{5+\dfrac{3}{x}-\dfrac{3}{x^2}}== \dfrac{\lim\limits_{x\rightarrow +\infty} \lheft(7-\dfrac{8}{x}+\dfrac{2}{x^2} \right)}{\lim\limits_{x\rightarrow +\infty}\lheft(5+\dfrac{3}{x}-\dfrac{3}{x^2}\right)}=\dfrac{7-0+0}{5+0-0}=\dfrac{7}{5}. 
\]
++++
.. latexmath:[$\lim\limits_{x\rightarrow -\infty}\dfrac{12-3x+6x^4}{1+x^6}$]
*Solução* Neste caso, dividimos o numerador e o denominador por latexmath:[$x^6$] e obtemos:
[latexmath]
++++
\[
\lim\limits_{x\rightarrow -\infty}\dfrac{12-3x+6x^4}{1+x^6}== \lim\limits_{x\rightarrow -\infty}\dfrac{\dfrac{12}{x^6}-\dfrac{3}{x^5}+\dfrac{6}{x^2}}{\dfrac{1}{x^6}+1}== \dfrac{\lim\limits_{x\rightarrow +\infty} \lheft( \dfrac{12}{x^6}-\dfrac{3}{x^5}+\dfrac{6}{x^2}\right)}{\lim\limits_{x\rightarrow +\infty}\lheft(\dfrac{1}{x^6}+1  \right)}=\dfrac{0-0+0}{0+1}=0.
\]
++++
.. latexmath:[$\lim\limits_{x\rightarrow +\infty}\dfrac{12x+6}{3-4x}$]

*Solução* Dividimos o numerador e o denominador por latexmath:[$x$] e obtemos:
[latexmath]
++++
\[
\lim\limits_{x\rightarrow +\infty}\dfrac{12x+6}{3-4x}== \lim\limits_{x\rightarrow -\infty}\dfrac{12+\dfrac{6}{x}}{\dfrac{3}{x}-4}== \dfrac{\lim\limits_{x\rightarrow +\infty} \lheft( 12+\dfrac{6}{x}\right)}{\lim\limits_{x\rightarrow +\infty}\lheft({\dfrac{3}{x}-4} \right)}=\dfrac{12+0}{0-4}=-3.
\]
++++

.. latexmath:[$\lim\limits_{x\rightarrow -\infty}\sqrt{x^2-2x+4}+x$]

*Solução* Para aplicar a metodologia dos exemplos anteriores, precisamos expressar a função como um quociente e para isso devemos racionalizar, isto é:
[latexmath]
++++
\[
\lim\limits_{x\rightarrow -\infty}\sqrt{x^2-2x+4}+x == \lim\limits_{x\rightarrow -\infty}\dfrac{\lheft(\sqrt{x^2-2x+4}+x\right) \lheft(\sqrt{x^2-2x+4}-x\right)}{\sqrt{x^2-2x+4}-x}=\lim\limits_{x\rightarrow -\infty}\dfrac{x^2-2x+4-x^2}{\sqrt{x^2-2x+4}-x} =\lim\limits_{x\rightarrow -\infty}\dfrac{-2x+4}{\sqrt{x^2-2x+4}-x} 
\]
++++
Desde que latexmath:[$x$] toma valores negativos bastante grandes, então dividimos por latexmath:[$x=-\sqrt{x^2}$], obtemos:
[latexmath]
++++
\[
\lim\limits_{x\rightarrow -\infty}\dfrac{-2x+4}{\sqrt{x^2-2x+4}-x}== \lim\limits_{x\rightarrow -\infty}\dfrac{-2+\dfrac{4}{x}}{-\sqrt{1-\dfrac{2}{x}+\dfrac{4}{x^2}}-1}=\dfrac{-2+0}{-\sqrt{1-0+0}-1}=1 
\]
++++
Portanto latexmath:[$\lim\limits_{x\rightarrow -\infty}\sqrt{x^2-2x+4}+x=1$].

.. latexmath:[$\lim\limits_{x\rightarrow +\infty}\dfrac{\sqrt{x+\sqrt{x+\sqrt{x+3}}}}{\sqrt{x+3}}$]

Solução:: Observamos que latexmath:[$x$] toma valores positivos, assim dividimos o numerador e o denominador por latexmath:[$\sqrt{x}$] e obtemos:
[latexmath]
++++
\[
\lim\limits_{x\rightarrow +\infty}\dfrac{\sqrt{x+\sqrt{x+\sqrt{x+3}}}}{\sqrt{x+3}}== \lim\limits_{x\rightarrow +\infty}\dfrac{\sqrt{1+\sqrt{\dfrac{1}{x}+\sqrt{\dfrac{1}{x^3}+\dfrac{3}{x^4}}}}}{\sqrt{1+\dfrac{3}{x}}}== \dfrac{\sqrt{1+\sqrt{0+\sqrt{0+0}}}}{\sqrt{1+0}}=1.
\]
++++

=== Limites infinitos
 Consideremos a função latexmath:[$f(x)=\dfrac{1}{x-1}$] a qual tem como gráfico:
%%colocar o gráfico do livro do espinoza trocando 2 por 1

No gráfico observamos que quando latexmath:[$x$] se aproxima para latexmath:[$1$] pela direita, a função latexmath:[$f(x)$] cresce sem limite em este caso denotamos por latexmath:[$\lim\limits_{x\rightarrow 1^{+}}f(x)=+\infty$], e no caso quando latexmath:[$x$] se aproxima para latexmath:[$1$] pela esquerda, a função latexmath:[$f(x)$] decresce sem limite e em este caso denotamos por latexmath:[$\lim\limits_{x\rightarrow 1^{+}}f(x)=+\infty$], a este tipo de limites são chamados de limites infinitos. A continuação os definiremos.

Definição::  Dizemos que o limite de latexmath:[$f(x)$] é latexmath:[$+\infty$] quando latexmath:[$x$] tende ao ponto latexmath:[$a$], e o denotamos por latexmath:[$\lim\limits_{x\rightarrow a}f(x)=+\infty$], se 
[latexmath]
++++
\[
\forall K>>0,\, \exists \delta>0\,:\, 0<|x-a|<\delta\,\Rightarrow\,f(x)>K.
\]
++++

Definição::  Dizemos que o limite de latexmath:[$f(x)$] é latexmath:[$-\infty$] quando latexmath:[$x$] tende ao ponto latexmath:[$a$], e o denotamos por latexmath:[$\lim\limits_{x\rightarrow a}f(x)=-\infty$], se 
[latexmath]
++++
\[
\forall M>>0,\, \exists \delta>0\,:\, 0<|x-a|<\delta\,\Rightarrow\,f(x)<-M.
\]
++++
No caso dos limites infinitos, também se definem os seguintes limites laterais:
[latexmath]
++++
\[
\lim\limits_{x\rightarrow a^{+}}f(x)=+\infty,\quad\lim\limits_{x\rightarrow a^{-}}f(x)=+\infty,\quad \lim\limits_{x\rightarrow a^{-}}f(x)=-\infty,\quad \lim\limits_{x\rightarrow a^{+}}f(x)=-\infty.
\]
++++
*Teorema* Se latexmath:[$n$] é um número inteiro positivo, então:
.. latexmath:[$\lim\limits_{x\rightarrow 0^{+}}\dfrac{1}{x^n}=+\infty$];
.. latexmath:[$\lim\limits_{x\rightarrow 0^{-}}\dfrac{1}{x^n}=
\lheft\{
\begin{array}{ccl}
-\infty,& & \mbox{ se }\,\, n\,\, \mbox{é impar}\\
+\infty,& & \mbox{ se }\,\, n\,\, \mbox{é par}
\end{array}\right.
$];

*Exemplos:* Alguns casos particulares do resultado anterior são:
[latexmath]
++++
\[
\lim\limits_{x\rightarrow 0^{+}}\dfrac{1}{x^5}=+\infty, \quad\lim\limits_{x\rightarrow 0^{+}}\dfrac{1}{x^4}=+\infty, \quad \lim\limits_{x\rightarrow 0^{-}}\dfrac{1}{x^3}=-\infty, \quad \lim\limits_{x\rightarrow 0^{+}}\dfrac{1}{x^6}=+\infty.
\]
++++

Definição::  Seja latexmath:[$f$] uma função com domínio latexmath:[${\rm Dom}(f)$].
.. Se latexmath:[${\rm Dom}(f)$] contem algum intervalo da forma latexmath:[$(a,+\infty)$], então:

.. (a) latexmath:[$\lim\limits_{x\rightarrow +\infty}f(x)=+\infty\Leftrightarrow \forall K\gg 0,\, \exists M>0\,:\, x>M\,\Rightarrow\,f(x)>K$].
.. (b) latexmath:[$\lim\limits_{x\rightarrow +\infty}f(x)=-\infty\Leftrightarrow \forall K\gg 0,\, \exists M>0\,:\, x>M\,\Rightarrow\,f(x)<-K$].
.. (c) latexmath:[$\lim\limits_{x\rightarrow -\infty}f(x)=+\infty\Leftrightarrow \forall K\gg 0,\, \exists M>0\,:\, x<-M\,\Rightarrow\,f(x)>K$].
.. (a) latexmath:[$\lim\limits_{x\rightarrow -\infty}f(x)=-\infty\Leftrightarrow \forall K\gg 0,\, \exists M>0\,:\, x<-M\,\Rightarrow\,f(x)<-K$].

A definição (a), significa que para os valores de latexmath:[$x$] bastante grandes (positivos), os valores correspondentes a latexmath:[$
f(x)$] também serão bastante grandes (positivos). Os itens (b),(c)e (d) são interpretados da mesma forma.

*Observação* 
Tendo-se em consideração que os símbolos latexmath:[$+\infty$] e latexmath:[$-\infty$] não são números reais, ningum dos limites "infinitos" existem. O termo "existe limite" sera usado somente quando o limite é um número real.

O seguinte resultado apresenta algumas propriedades que nos permitem calcular limites infinitos.

*Propriedades* Seja latexmath:[$a\in \mathbb{R}$] tal que latexmath:[$\lim\limits_{x\rightarrow a}f(x)=0$] e latexmath:[$\lim\limits_{x\rightarrow a}g(x)=c$], onde latexmath:[$c\neq 0$].
.. Se latexmath:[$c>0$] e latexmath:[$f(x)\rightarrow 0$], através de valores positivos  latexmath:[$\Rightarrow \lim\limits_{x\rightarrow a}\dfrac{g(x)}{f(x)}=+\infty$].
.. Se latexmath:[$c>0$] e latexmath:[$f(x)\rightarrow 0$], através de valores negativos  latexmath:[$\Rightarrow \lim\limits_{x\rightarrow a}\dfrac{g(x)}{f(x)}=-\infty$].
.. Se latexmath:[$c<0$] e latexmath:[$f(x)\rightarrow 0$], através de valores positivos  latexmath:[$\Rightarrow \lim\limits_{x\rightarrow a}\dfrac{g(x)}{f(x)}=-\infty$].
.. Se latexmath:[$c<0$] e latexmath:[$f(x)\rightarrow 0$], através de valores negativos  latexmath:[$\Rightarrow \lim\limits_{x\rightarrow a}\dfrac{g(x)}{f(x)}=+\infty$].
Alhem das anteriores consideraremos outras propriedades para calcular os limites infinitos:

*Teorema* Seja [$f$] e latexmath:[$g$] duas funções

.. Se latexmath:[$\lim\limits_{x\rightarrow \pm\infty} f(x) =\pm\infty$] e latexmath:[$\lim\limits_{x\rightarrow \pm\infty} g(x) =\pm\infty$], então latexmath:[$\lim\limits_{x\rightarrow \pm\infty}( f(x)+g(x)) =\pm\infty$] e latexmath:[$\lim\limits_{x\rightarrow \pm\infty}( f(x)\cdot g(x)) =\pm\infty$]; 
.. Se latexmath:[$\lim\limits_{x\rightarrow \pm\infty} f(x) =L$] e latexmath:[$\lim\limits_{x\rightarrow \pm\infty} g(x) =\pm\infty$], então latexmath:[$\lim\limits_{x\rightarrow \pm\infty}( f(x)+g(x)) =\pm\infty$]; 
.. Se latexmath:[$\lim\limits_{x\rightarrow \pm\infty} f(x) =L$], latexmath:[$L>0$], e latexmath:[$\lim\limits_{x\rightarrow \pm\infty} g(x) =\pm\infty$], então  latexmath:[$\lim\limits_{x\rightarrow \pm\infty}( f(x)\cdot g(x)) =\pm\infty$];
.. Se latexmath:[$\lim\limits_{x\rightarrow \pm\infty} f(x) =L$], latexmath:[$L<0$], e latexmath:[$\lim\limits_{x\rightarrow \pm\infty} g(x) =\pm\infty$], então  latexmath:[$\lim\limits_{x\rightarrow \pm\infty}( f(x)\cdot g(x)) =\mp\infty$];
.. Se latexmath:[$\lim\limits_{x\rightarrow \pm\infty} f(x) =\mp\infty$] e latexmath:[$\lim\limits_{x\rightarrow \pm\infty} g(x) =\pm\infty$], então  latexmath:[$\lim\limits_{x\rightarrow \pm\infty}( f(x)\cdot g(x)) =\mp\infty$];
.. Se latexmath:[$\lim\limits_{x\rightarrow \pm\infty} f(x) =L$],  onde latexmath:[$L$] é constante, e latexmath:[$\lim\limits_{x\rightarrow \pm\infty} g(x) =\pm\infty$], então  latexmath:[$\lim\limits_{x\rightarrow \pm\infty}\dfrac{f(x)}{g(x)} =\pm\infty$].

*Observação* O teorema anterior pode ser resumido da seguinte forma, latexmath:[$k$] é uma constante:
.. latexmath:[$ k + (+\infty) =+\infty$]
.. latexmath:[$ k + (-\infty) =-\infty$]
.. latexmath:[$ (+\infty) + (+\infty) =+\infty$]
.. latexmath:[$ (-\infty) + (-\infty) =-\infty$]
.. latexmath:[$ (+\infty) (+\infty) =+\infty$]
.. latexmath:[$ (-\infty) (-\infty) =+\infty$]
.. latexmath:[$ (+\infty) (-\infty) =-\infty$]
.. latexmath:[$ \dfrac{k}{\pm\infty} =0$]
.. latexmath:[$(-\infty)^n=
\lheft\{
\begin{array}{ccl}
+\infty,& & \mbox{ se }\,\, n\,\, \mbox{é par positivo}\\
-\infty,& & \mbox{ se }\,\, n\,\, \mbox{é impar positivo}
\end{array}\right.
$];
.. latexmath:[$(+\infty)^n=+\infty$], latexmath:[$n\in\mathbb{Z}^{+}$]
.. latexmath:[$k(+\infty)=

.{zwsp}==== Calcule latexmath:[$\lim\limits_{x\rightarrow 1^{-}} f(x)$] e latexmath:[$\lim\limits_{x\rightarrow 1^{+}} f(x)$] se latexmath:[$
f(x)=\dfrac{4x^3 -1}{2-x-x^2}$].

Solução:: Ao avaliar latexmath:[$f(x)$] no ponto latexmath:[$x=1$], observamos que temos a forma latexmath:[$\dfrac{3}{0}$], aplicando as propriedades podemos concluir que os dois limites são infinitos. Para determinar o sinal de infinito, fatoramos o denominador para analisar se se aproxima a latexmath:[$0$] por valores positivos ou negativos. Assim:
.. latexmath:[$\lim\limits_{x\rightarrow 1} (4x^3 -1)=3>0$].
.. latexmath:[$\lim\limits_{x\rightarrow 1} (2-x-x^2)=\lim\limits_{x\rightarrow 1} (1-x)(x+2)$].
.. (a) Para latexmath:[$x<1$] (muito próximo a latexmath:[$1$]): latexmath:[$1-x>0$] e latexmath:[$x+2>0$] então latexmath:[$\lim\limits_{x\rightarrow 1^{-}} (1-x)(x+2)=0^{+}$], isto é, se aproxima ao zero por valores positivos.

.. (b) Para latexmath:[$x>1$] (muito próximo a latexmath:[$1$]): latexmath:[$1-x<0$] e latexmath:[$x+2>0$] então latexmath:[$\lim\limits_{x\rightarrow 1^{+}} (1-x)(x+2)=0^{-}$], isto é, se aproxima ao zero por valores negativos.

Portanto
[latexmath]
++++
\[
\lim\limits_{x\rightarrow 1^{-}} \dfrac{4x^3 -1}{2-x-x^2} == \frac{3}{0^{+}}=+\infty\,\,\mbox{e}\,\,\lim\limits_{x\rightarrow 1^{+}} \dfrac{4x^3 -1}{2-x-x^2} == \frac{3}{0^{-}}=-\infty.
\]
++++

*Exemplos* Calcular os seguintes limites:

.. latexmath:[$\lim\limits_{x\rightarrow 2^{+}} \dfrac{x+2}{x^2-4}$]
Solução:: 
latexmath:[$\lim\limits_{x\rightarrow 2^{+}} \dfrac{x+2}{x^2-4}=\lim\limits_{x\rightarrow 2^{+}} \dfrac{x+2}{(x-2)(x+2)}=\lim\limits_{x\rightarrow 2^{+}} \dfrac{1}{x-2=+\infty}$]

.. latexmath:[$\lim\limits_{x\rightarrow 4^{-}} \dfrac{\sqrt{16-x^2}}{x-4}$]
Solução::
latexmath:[$\lim\limits_{x\rightarrow 4^{-}} \dfrac{\sqrt{16-x^2}}{x-4}=\lim\limits_{x\rightarrow 4^{-}} \dfrac{16-x^2}{(x-4)\sqrt{16-x^2}}== \lim\limits_{x\rightarrow 4^{-}} \dfrac{(4-x)(4+x)}{(x-4)\sqrt{16-x^2}}== -\lim\limits_{x\rightarrow 4^{-}} \dfrac{4+x}{\sqrt{16-x^2}} =\dfrac{-8}{0^{+}}=-\infty$].

.. latexmath:[$\lim\limits_{x\rightarrow 4^{-}} \dfrac{\lheft\lfloor x\right\rfloor-4}{x-4}$]
Solução:: Observemos que latexmath:[$x\rightarrow 4^{-}\Rightarrow x\in [3,4)\Rightarrow \lheft\lfloor x\right\rfloor =3$] então:
latexmath:[$\lim\limits_{x\rightarrow 4^{-}} \dfrac{\lheft\lfloor x\right\rfloor-4}{x-4}== \lim\limits_{x\rightarrow 4^{-}} \dfrac{3-4}{x-4} =\lim\limits_{x\rightarrow 4^{-}} \dfrac{-1}{x-4} == \dfrac{-1}{0^{-}}=+\infty$].

Seja latexmath:[$ P(x)=a_0 x^n +a_1 x^{n-1} +\cdots +a_n$] um polinômio de grau latexmath:[$n$] e latexmath:[$ Q(x)=b_0 x^m +b_1 x^{m-1} +\cdots +b_m$] um polinômio de grau latexmath:[$m$], então
.. latexmath:[$\lim\limits_{x\rightarrow \pm\infty} P(x)=\lim\limits_{x\rightarrow \pm\infty}(a_0 x^n +a_1 x^{n-1} +\cdots +a_n) == \lim\limits_{x\rightarrow \pm\infty} a_0 x^n$]

\infty & &\mbox{se}\,\, n>m;\\

0 & &\mbox{se}\,\, n<m;
\end{array}\right.
$]

=== Assintotas


Definição:: 
Diz-se que a reta latexmath:[$L$] é uma *assintota* da curva latexmath:[$C$] se a distancia entre a reta  latexmath:[$L$] e o ponto latexmath:[$A$], que se movimenta ao longo da curva, tende a zero quando latexmath:[$A$] tende ao infinito, isto é latexmath:[$\lim\limits_{A\rightarrow \infty} {\rm Dist}(L,A)=0$]. 

Proposição:: A reta latexmath:[$x=a$] é uma assintota vertical do gráfico latexmath:[$ y=f(x)$] se verifica-se uma das seguintes condições:
[latexmath]
++++
\[
\mbox{(a)}\,\lim\limits_{x\rightarrow a} f(x)=\pm\infty,\quad \mbox{(b)}\,\lim\limits_{x\rightarrow a^{+}} f(x)=\pm\infty,\quad \mbox{(c)}\,\lim\limits_{x\rightarrow a^{-}} f(x)=\pm\infty.
\]
++++
%%%%%%%%Modificar os gráficos do mitacc 3.21,3.22,3.23

Proposição:: A reta latexmath:[$y=k$] é uma assintota horizontal do gráfico latexmath:[$ y=f(x)$] se se verifica uma das seguintes condições:
[latexmath]
++++
\[
\mbox{(a)}\,\lim\limits_{x\rightarrow +\infty} f(x)=k,\quad \mbox{(b)}\,\lim\limits_{x\rightarrow -\infty} f(x)=k.
\]
++++
%%%%%%%%Modificar os gráficos do mitacc 3.24, 3.25


Proposição:: A reta latexmath:[$y=mx+b,\,\,m\neq0$] é uma assintota obliqua do gráfico latexmath:[$ y=f(x)$] se e somente se uma das seguintes condições é verificada:
.. latexmath:[$\lim\limits_{x\rightarrow +\infty} \dfrac{f(x)}{x}=m$] e latexmath:[$\lim\limits_{x\rightarrow +\infty} (f(x)-mx)=b$];
.. latexmath:[$\lim\limits_{x\rightarrow -\infty} \dfrac{f(x)}{x}=m$] e latexmath:[$\lim\limits_{x\rightarrow -\infty} (f(x)-mx)=b$].

%%%%%%%%Modificar os gráficos do mitacc 3.26, 3.27

[NOTE]
==== 
.. Se ao calcular os valores de latexmath:[$m$] e latexmath:[$b$] (quando latexmath:[$x\rightarrow +\infty$]), um dos limites não existe, a curva não apresenta assintota obliqua à direita. Da mesma forma obtemos quando latexmath:[$x\rightarrow -\infty$]. 
.. Se latexmath:[$m=0$] e latexmath:[$b$] finito, a assintota é horizontal.
.. Se uma função latexmath:[$f(x)$] tem a forma de uma fração, as possíveis assintotas verticais se obtêm nos valores de latexmath:[$x$] que anulam ao denominador de latexmath:[$f(x)$]. Uma vez encontrados estos valores, devemos comprovar se seu limite é infinito.
==== 

.{zwsp}
====
Encontrar as assintotas da função latexmath:[$ y=f(x)$] 
.. latexmath:[$ f(x)=\dfrac{x^2+9}{x-3}$] 

Solução::  
Observamos que o denominador se anula para latexmath:[$x=3$] então latexmath:[$\lim\limits_{x\rightarrow 3}\dfrac{x^2+9}{x-3}=\pm\infty$], assim latexmath:[$x=3$] é uma assintota vertical.

Calculemos as assintotas horizontais caso existam: latexmath:[$y=k$] onde latexmath:[$k=\lim\limits_{x\rightarrow \infty}\dfrac{x^2+9}{x-3}=+\infty$], portanto não existem assintotas horizontais.

Calculemos as assintotas oblíquas: latexmath:[$y=mx+b$] onde: latexmath:[$m=\lim\limits_{x\rightarrow \pm\infty}\dfrac{f(x)}{x}=\lim\limits_{x\rightarrow \infty}\dfrac{x^2+9}{x^2-3x}=1$] obtemos latexmath:[$m=1$], latexmath:[$b=\lim\limits_{x\rightarrow \pm\infty} (f(x)-mx)=\lim\limits_{x\rightarrow \infty}\dfrac{x^2+9}{x-3}-x=\lim\limits_{x\rightarrow \infty}\dfrac{3x+9}{x-3}=3$] obtemos latexmath:[$b=3$]. Portanto a assintota vertical é a reta latexmath:[$ y=x+3$].
