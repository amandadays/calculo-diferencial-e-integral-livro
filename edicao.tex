=== Composição de funções

Definição::
Sejam latexmath:[$f:A\rightarrow B$] e latexmath:[$f:B\rightarrow C$] duas funções reais tais que latexmath:[${\rm Im}(f)\cap {\rm Dom}(g)\not=\emptyset$]. A *composição* de latexmath:[$g$] com latexmath:[$f$], denotada por latexmath:[$g\circ f$], é a função latexmath:[$g\circ f : A\rightarrow C$] definida por:
+
[latexmath]
++++
\[
(g\circ f)(x):=g(f(x))
\]
++++ 
O domínio da função composta latexmath:[$g\circ f$] é dado por 
[latexmath]
++++
\[
{\rm Dom}(g\circ f) = \left\{ x\in \mathbb{R}\,:\, x\in  {\rm Dom}(f) \mbox{ e }  f(x)\in{\rm Dom}(g) \right\}
\]
++++
Na figura seguinte ilustramos a função composta latexmath:[$g\circ f$]

%%funcao composta


*Exemplo* Sejam latexmath:[$f(x)=2x-6$] e latexmath:[$g(x)=\sqrt{x}$] duas funções reais. Encontre  latexmath:[$g\circ f$] e  latexmath:[$f\circ g$]
*Solução* 
.. latexmath:[$(g\circ f)(x)=g(f(x))= g\left(2x-6\right)=\sqrt{2x-6} $], o domínio da  latexmath:[$g\circ f$] é
[latexmath]
++++
\[
\begin{array}{rcl}
{\rm Dom}(g\circ f) &=& \left\{ x\in \mathbb{R}\,:\, x\in  {\rm Dom}(f) \mbox{ e }  f(x)\in{\rm Dom}(g) \right\}\\
                    &=& \left\{ x\in \mathbb{R}\,:\, x\in   \mathbb{R} \mbox{ e }  2x-6\geq 0 \right\}\\
										&=& [3,+\infty)
\end{array}
\]
++++
.. latexmath:[$(f\circ g)(x)=f(g(x))= f\left(\sqrt{x}\right)=2\sqrt{x}-6$], o domínio da  latexmath:[$f\circ g$] é
[latexmath]
++++
\[
\begin{array}{rcl}
{\rm Dom}(f\circ g) &=& \left\{ x\in \mathbb{R}\,:\, x\in  {\rm Dom}(g) \mbox{ e }  g(x)\in{\rm Dom}(f) \right\}\\
                    &=& \left\{ x\in \mathbb{R}\,:\, x\geq 0 \mbox{ e }  \sqrt{x}\in \mathbb{R} \right\}\\
										&=& [0,+\infty)
\end{array}
\]
++++
Assim concluirmos que a composição de funções não é comutativa, isto é,  latexmath:[$g\circ f$] e  latexmath:[$f\circ g$] são geralmente diferentes.

*Exemplo* Encontre  latexmath:[$f\circ g$] onde:
[latexmath]
++++
\[
f(x)=\left\{
\begin{array}{rcl} 
x^2 & &\mbox{ se } x<1;\\
-x^3& &\mbox{ se } x\geq 2
\end{array}
\right.
\quad
g(x)=\left\{
\begin{array}{rcl} 
-x & &\mbox{ se } x<2;\\
2x& &\mbox{ se } x\geq 4
\end{array}
\right.
\]
++++
*Solução* neste caso cada uma das funções tem duas regras de correspondência:
[latexmath]
++++
\[
f(x)=\left\{
\begin{array}{rcl} 
f_1 & &\mbox{ se } x\in {\rm Dom}(f_1);\\
f_2& &\mbox{ se } x\in {\rm Dom}(f_2)
\end{array}
\right.
\quad
g(x)=\left\{
\begin{array}{rcl} 
g_1 & &\mbox{ se } x\in {\rm Dom}(g_1);\\
g_2& &\mbox{ se } x\in {\rm Dom}(g_2)
\end{array}
\right.
\]
++++
O domínio de latexmath:[$f\circ g$] obtemos analisando as combinações possíveis, isto é:
.. latexmath:[$f_1\circ g_1$]:
[latexmath]
++++
\[
\begin{array}{rcl}
{\rm Dom}(f_1\circ g_1) &=& \left\{ x\in \mathbb{R}\,:\, x\in  {\rm Dom}(g_1) \mbox{ e }  g_1(x)\in{\rm Dom}(f_1) \right\}\\
                    &=& \left\{ x\in \mathbb{R}\,:\, x\in (-\infty,2) \mbox{ e }  -x\in (-\infty,1) \right\}\\
										&=& \left\{ x\in \mathbb{R}\,:\, x\in (-\infty,2) \mbox{ e }  x\in (-1,+\infty) \right\}\\
										&=& (-1,2)
\end{array}
\]
++++
Neste caso a composta esta definida por latexmath:[$(f_1\circ g_1)(x)=f_1(g_1(x))=f_1(-x)=x^2$].
.. latexmath:[$f_1\circ g_2$]:
[latexmath]
++++
\[
\begin{array}{rcl}
{\rm Dom}(f_1\circ g_2) &=& \left\{ x\in \mathbb{R}\,:\, x\in  {\rm Dom}(g_2) \mbox{ e }  g_2(x)\in{\rm Dom}(f_1) \right\}\\
                    &=& \left\{ x\in \mathbb{R}\,:\, x\in [4,+\infty) \mbox{ e }  2x\in (-\infty,1) \right\}\\
										&=& \left\{ x\in \mathbb{R}\,:\, x\in [4,+\infty) \mbox{ e }  x\in (-\infty,\frac{1}{2}) \right\}\\
										&=& \emptyset
\end{array}
\]
++++
.. latexmath:[$f_2\circ g_1$]:
[latexmath]
++++
\[
\begin{array}{rcl}
{\rm Dom}(f_2\circ g_1) &=& \left\{ x\in \mathbb{R}\,:\, x\in  {\rm Dom}(g_1) \mbox{ e }  g_1(x)\in{\rm Dom}(f_2) \right\}\\
                    &=& \left\{ x\in \mathbb{R}\,:\, x\in (-\infty,2) \mbox{ e }  -x\in [2,+\infty) \right\}\\
										&=& \left\{ x\in \mathbb{R}\,:\, x\in (-\infty,2) \mbox{ e }  x\in (-\infty,-2] \right\}\\
										&=& (-\infty,-2]
\end{array}
\]
++++
Neste caso a composta esta definida por latexmath:[$(f_2\circ g_1)(x)=f_2(g_1(x))=f_2(-x)=x^3$].
.. latexmath:[$f_2\circ g_2$]:
[latexmath]
++++
\[
\begin{array}{rcl}
{\rm Dom}(f_2\circ g_2) &=& \left\{ x\in \mathbb{R}\,:\, x\in  {\rm Dom}(g_2) \mbox{ e }  g_2(x)\in{\rm Dom}(f_2) \right\}\\
                    &=& \left\{ x\in \mathbb{R}\,:\, x\in [4,+\infty) \mbox{ e }  2x\in [2,+\infty) \right\}\\
										&=& \left\{ x\in \mathbb{R}\,:\, x\in [4,+\infty) \mbox{ e }  x\in [1,+\infty) \right\}\\
										&=& [4,+\infty)
\end{array}
\]
++++
Neste caso a composta esta definida por latexmath:[$(f_2\circ g_2)(x)=f_2(g_2(x))=f_1(2x)=-8x^3$], portanto
[latexmath]
++++
\[
(f\circ g)(x)=\left\{
\begin{array}{rcl} 
(f_1\circ g_1)(x) & &\mbox{ se } x\in (-1,2);\\
(f_2\circ g_1)(x) & &\mbox{ se } x\in (-\infty,-2];\\
(f_2\circ g_2)(x) & &\mbox{ se } x\in [4,+\infty)
\end{array}
\right.
\quad
(f\circ g)(x)=\left\{
\begin{array}{rcl} 
x^3 & &\mbox{ se } x\in (-1,2);\\
x^2 & &\mbox{ se } x\in (-\infty,-2];\\
-8x^3 & &\mbox{ se } x\in [4,+\infty)
\end{array}
\right.
\]
++++

== Propriedades da composição de funções

Sejam latexmath:[$f,g$] e latexmath:[$h$] funções reais com domínios latexmath:[${\rm Dom}(f)$], latexmath:[${\rm Dom}(g)$] e latexmath:[${\rm Dom}(h)$], respectivamente. Então se verifica:
.. latexmath:[$(f\circ g)\circ h = f\circ (g\circ h)$]
.. latexmath:[$f\circ {\rm Id}= f ={\rm Id}\circ f$]
.. latexmath:[$(f+ g)\circ h = f\circ h +g\circ h$] 
.. latexmath:[$(f- g)\circ h = f\circ h -g\circ h$]
.. latexmath:[$(f\cdot g)\circ h = (f\circ h)\cdot (g\circ h)$]
.. latexmath:[$\left( \frac{f}{g}\right)\circ h = \frac{f\circ h}{g\circ h}$]

== Função inversa
Dada uma função latexmath:[$ f: A\rightarrow B$], acontece uma das seguintes possibilidades: latexmath:[$f$] é injetiva ou latexmath:[$f$] não é injetiva.
.. Se latexmath:[$f$] não é injetiva, existem pele menos dois elementos latexmath:[$x_1,x_2\in A$] tais que latexmath:[$(x_1,y)\in f$] e latexmath:[$(x_2,y)\in f$], portanto, a (relação) inversa de latexmath:[$f$] não é uma função de latexmath:[$B$] em latexmath:[$A$].

.. Se latexmath:[$ f: A\rightarrow B$] é injetiva, então a inversa latexmath:[$ f^{-1}: B\rightarrow A$] é uma função injetiva e é dita a *função inversa* de latexmath:[$f$]

==Propriedades da função inversa

Sejam latexmath:[$f$] e sua respectiva função inversa latexmath:[$f^{-1}$]. Então se verifica:
.. latexmath:[${\rm Dom}(f^{-1})={\rm Im}(f)$] e latexmath:[${\rm Im}(f^{-1})={\rm Dom}(f)$]
.. Uma função latexmath:[$f$] tem função inversa se, e somente se, latexmath:[$f$] é injetiva. Alem disso, se verifica:
.. i) latexmath:[$(f^{-1}\circ f)(x)=x$], para todo latexmath:[$x\in {\rm Dom}(f)$]
.. ii) latexmath:[$(f\circ f^{-1})(y)=y$], para todo latexmath:[$y\in {\rm Dom}(f^{-1})$]

.. Se latexmath:[$f$] é uma função real que tem função inversa, então os gráficos de latexmath:[$y=f(x)$] e latexmath:[$y=f^{-1}(x)$] são simétricas com respeito à reta bissetriz latexmath:[$y=x$]. ver figura (2.62).

.. Sejam as funções latexmath:[$f$], latexmath:[$g$] injetivas. Se existe latexmath:[$g\circ f$], então latexmath:[$(g\circ f)^{-1}= f^{-1}\circ g^{-1}$].

*Observação* Seja latexmath:[$f$] é uma função real definida por latexmath:[$y=f(x)$] a qual tem função inversa. Para encontrar a regra de correspondência da latexmath:[$f^{-1}$], colocamos o  latexmath:[$x$] em evidencia em termos da variável latexmath:[$y$]. Assim obtemos latexmath:[$x=f^{-1}(y)$]; porem a costume de representar com a variável latexmath:[$x$] a variável independe e com latexmath:[$y$] a a variável dependente, faz que escrevamos como latexmath:[$y=f^{-1}(x)$], trocando as variáveis latexmath:[$x$] e latexmath:[$y$] em latexmath:[$x=f^{-1}(y)$].
*Exemplo* Encontre a função inversa da função latexmath:[$ f(x)=5x-3$] se latexmath:[$x\in[0,6]$].

*Solução:* Verificamos que latexmath:[$f(x_1)=f(x_2)\Rightarrow 5x_1-3=5x_2-3 \Rightarrow x_1=x_2$], assim latexmath:[$f$] é injetiva. por outro lado, desde que latexmath:[$y=f(x)$], então latexmath:[$y=5x-3$], latexmath:[$x\in [0,6]$]. Despejamos a variável latexmath:[$x$]: latexmath:[$x=\frac{y+3}{5}$], para latexmath:[$x\in [0,6]$], logo determinamos a variação da variável  latexmath:[$y$]:
[latexmath]
++++
\[
x=\frac{y+3}{5}\in [0,6] \Rightarrow 0\leq \frac{y+3}{5} \leq 6 \Rightarrow 0\leq y +3 \leq 30 \Rightarrow -3\leq y \leq 27  \Rightarrow y\in[-3,27] 
\]
++++
Assim latexmath:[$x=\frac{y+3}{5}$], para latexmath:[$y\in [-3,27]$], permutamos latexmath:[$x$] por latexmath:[$y$] isto é latexmath:[$y=\frac{x+3}{5}$], para latexmath:[$x\in [-3,27]$]. Portanto latexmath:[$f^{-1}(x)=\frac{x+3}{5}$], para latexmath:[$x\in [-3,27]$].
